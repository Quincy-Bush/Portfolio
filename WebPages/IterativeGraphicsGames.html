This page contains numerous projects that I've completed for a UMN course, 4611: programming iterative graphics and games.
With these projects I've developed an understanding of concepts and algorithms necessary for programming computer graphics.
This includes identifying and solving issues with multi-dimensional graphics, writing shader programs, implementing real time computer-graphics and user interfaces for graphics interactions.
This required:

  • Knowledge and experience in TypeScript to expand on JavaScript variable type definitions, function parameters and return values to allow classes, objects and systems to be defined and implemented.
  • The ability to learn and adapt quickly to UMN computer graphics software library, GopherGfx, to analyze, create and edit computer graphics through the JavaScript API webGL.
  • Experience in HTML for canvas and element packaging, styling or scripting in the user interface, and scene environment

Text Rain: Interactive Art installation

  • This project focuses on using the GopherGfx TypeScript library to accept, interpet and edit 2D graphic, image space, and scene space coordinates to deliver a version of Camille Utterback's interactive art installation.
  • It works by building a simple 2D object that will have the GopherGfx's text class applied as its texture and then interprets user inputs via camera or video file, and updates the coordinates of the text object based on changes in color values..
Feel free to click the link here and try it for yourself!

Hole in the Ground: interactive 3D video game

  • This project uses the GopherGfx library with Typescript to build an interactive 3D graphics program inspired by a 2018 physics puzzle game Donut County
  • This program calculates 3D points, vectors, and geometric transformations to determine the proper logic for interaction with the game enviorment and physics between 3 dimension objects.
Feel free to click the link here and try it for yourself!

Earthquake Visualization: Geospatial data visualization

  • This project uses the GopherGfx library to build 3 dimensional program simulating location and magnitude of earthquakes across history on a 2D map capable of being transformed into a 3D globe through the user interface.
  • The program applies texture mapping to 3D objects to create the map and globe visualization while algorithms work to convert spherical coordinates to cartesianal, define vectors and texture coordiantes and reformation of 3D polygon models based on interaction with the user interface.
Feel free to click the link here and try it for yourself!

Artistic Rendering: Real-time GLSL shading

  • This project uses the GopherGfx library to calculate and render a wide range of lighting effects using GLSL in real time based on selections made in the user interface.
  • This program works to calculate, create and modify viewpoint based lighting effects, specifically lighting and normals per-pixel values to determine the complexity of the surface and interaction with lighting.
