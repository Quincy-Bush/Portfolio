Interactive Computer graphics

Programming Interactive Computer Graphics and Games| Fall 2023
Quincy.m.bush@gmail.com 507-884-9868 LinkedIn
Overview
  • This page contains numerous projects I created for the completion of University of Minnesota course, 4611: programming interactive graphics and games.
    Below you can find details and links to download and view each project in action.
Projects
Text Rain: Interactive Art installation
  • This project focuses on using TypeScript to build, interpret and edit 2D graphic, image space, and scene space coordinates to deliver a version of Camille Utterback's interactive art installation.
  • It works by building a simple 2D object with GopherGfx's text class as its texture, then interpreting user inputs via camera or video file, and updating the coordinates of the text object based on color value changes.
    • Download Interpretation

Text Rain Implementation

An interactive graphics program using changes in color values to simulate falling objects masked as letters as raindrops.

Hole in the Ground: interactive 3D video game
  • This project uses Typescript to build an interactive 3D graphics program inspired by a 2018 physics puzzle game Donut County
  • This program calculates 3D points, vectors, and geometric transformations to determine the proper logic needed for interaction with the game enviroment and physics between 3-dimension objects.
    • Download Interpretation

Hole-in-the Ground

A simple 3-dimensional graphics program demonstrating the use of 3D points, vectors, and geometric transformations in a game environment.

Earthquake Visualization: Geospatial data visualization
  • This project builds a 3-dimensional simulation of the location and magnitude of earthquakes throughout history, displayed on a 2D map that can be transformed into a 3D globe via the user interface.
  • The program applies texture mapping to 3D objects to create the map and globe visualization, while algorithms convert spherical coordinates to cartesian, define vectors and texture coordinates, and reform 3D polygon models based on user interaction.
    • Download Interpretation

Earthquake Visualization

A 3-dimensional graphics program utilizing data visualization and recursive algorithms to display the location and magnitude of earthquakes throughout history.

Artistic Rendering: Real-time GLSL shading
  • This program renders a wide range of lighting effects, models, and textures in real time, based on selections made in the user interface.
  • This program calculates, creates, and modifies 3D model representations based on lighting effects, specifically using per-pixel lighting and normals to determine surface shading and complexity via polygons and their interaction with lighting.
    • Download Interpretation

Artistic Rendering Project

A 3 dimensional Graphics program allowing changes to lighting, textures and models through a user interface.

Key Focuses and Details

verall, these projects provide a foundation of concepts and algorithms necessary for programming interactive computer graphics.

  • Identifying and solving issues with multi-dimensional graphics, writing shader programs, implementing real time computer-graphics and user interfaces for graphical interactions.
  • Knowledge and experience in TypeScript to expand on JavaScript variable type definitions, function parameters, and return values, allowing classes, objects, and systems to be defined and implemented.
  • The ability to learn and adapt quickly to UMN computer graphics software library, GopherGfx, to analyze, create and edit computer graphics through the JavaScript API webGL.
  • Experience in HTML for canvas and element packaging, styling or scripting in the user interface, and scene environment.
